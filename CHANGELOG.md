# Changelog

## [0.1.0] — Primeira versão pública

### Adicionado

- 26 regras de acessibilidade cobrindo WCAG 2.1 nível A e AA
- Suporte a arquivos HTML, JSX e TSX
- Parser AST (parse5 para HTML, @babel/parser para JSX/TSX)
- Quick Fix para regras com correção determinística
- Hover com documentação WCAG ao passar o mouse em um aviso
- Status bar com contagem de problemas no arquivo atual
- Output channel com log de execução por arquivo
- Configuração por regra via `a11yDevHelper.rules` nas settings
